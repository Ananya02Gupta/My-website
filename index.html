About This Website

This website is created as a part of the Responsive Web Design task. It demonstrates the use of modern CSS techniques such as Flexbox and CSS Grid to build a flexible and responsive layout.

Flexbox is used for arranging and aligning elements like the header and navigation menu, while CSS Grid is used to organize the image gallery in a clean and structured manner.

The website is fully responsive and adjusts automatically to different screen sizes such as desktop, tablet, and mobile devices using media queries.
